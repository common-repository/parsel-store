== Description ==

Creates a Parsel store on your wordpress blog. You have to have a Parsel store in order to add it to your wordpress blog.
If you currently do not have a Parsel store and would like to get one please email info@parsel.me.

== Installation ==

= Install the plugin =

1. Upload plugin
1. Visit "Parsel Store" menu item
1. Fill in your Parsel store ID
1. Fill optional Google Analytics code to track statistics
1. C yolick "Save changes" andu are done.

= Use the plugin =

1. Create a new page in wordpress blog. Preferebly a page with no side columns, this way the store will be on a stand-
alone page.
1. In the page content add [parse] shortcode. The store will appear on that page whereever you add this shortcode.
